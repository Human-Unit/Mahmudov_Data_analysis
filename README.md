Анализ данных о ментальном здоровье подростков
Данный проект посвящен исследованию взаимосвязи между уровнями тревожности, сна и стресса у подростков с использованием методов статистического анализа и машинного обучения.

Основные возможности
Расчет статистических метрик: Вычисление среднего значения, среднего отклонения и средней абсолютной ошибки (MAE).

Корреляционный анализ: Определение силы взаимосвязи между различными показателями датасета.

Линейная регрессия: Математическая модель для предсказания уровня стресса на основе входных данных.

Визуализация: Построение графиков рассеяния и линий регрессии для наглядного представления трендов.

Используемые технологии
Python

Pandas для обработки данных

Matplotlib для построения графиков

Scikit-learn для базовых алгоритмов машинного обучения

Как использовать
Убедитесь, что файл Teen_Mental_Health_Dataset.csv находится в корневой папке проекта.

Установите необходимые библиотеки через терминал: pip install pandas matplotlib scikit-learn.

Запустите основной скрипт для получения аналитических отчетов и графиков.

Особенности реализации
Все ключевые алгоритмы, включая расчет коэффициента корреляции и параметров линейной регрессии, реализованы на чистом Python без использования встроенных функций библиотек, где это было необходимо для понимания математической логики процесса.
