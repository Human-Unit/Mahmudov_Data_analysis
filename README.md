Teen Mental Health Analysis

Анализ данных о ментальном здоровье подростков (взаимосвязь тревожности, сна и стресса).

Базовые статистические алгоритмы (корреляция, MAE, линейная регрессия) написаны с нуля на чистом Python без использования готовых библиотек, не хотел использовать готовые библиотеки. Под конец добавил простую линейную регрессию, но из-за целых чисел график выглядит так себе.

Файлы

analysis.ipynb — основной Jupyter-ноутбук с расчетами и графиками.

Teen_Mental_Health_Dataset.csv — исходный датасет.

Есть также два старых питоновских файла, один из которых полностью сгнерирован ИИ для проверки данных полученных моим кодом.
Далее перешёл на jupiter.

Запуск

Установите зависимости:

pip install pandas matplotlib scikit-learn


Откройте и запустите analysis.ipynb в PyCharm, VS Code или Jupyter.
